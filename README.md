## üöÄ Key Features

### üåê NetWatch (Network Intelligence)
* **Real-time Ping Monitoring:** Live status tracking for Servers, Routers, CCTV, etc.
* **Speedtest Integration:** Automatic background internet speed checks (Download/Upload/Ping).
* **Instant Alerts:** Sends **Telegram Notifications** immediately when a device goes DOWN or UP.
* **Secure Access:** Encrypted Login System with User Management.

### üíª SysGaze (Hardware Telemetry)
* **Multi-Drive Analysis:** Automatically detects and monitors all storage drives (C:, D:, F:, etc.).
* **Resource Tracking:** Live CPU & RAM usage visualization.
* **Uptime Monitor:** Tracks system active duration accurately.

### üßπ The Cleaner (Automated Maintenance)
* **Silent Background Process:** Runs invisibly to clean temporary files and cache.
* **Storage Optimization:** Keeps the system running smoothly without user intervention.
